# TERMUX-LOCK
Fingerprint lock in termux.

## INSTALL

````bash
pkg update && upgrade
pkg install git -y
git clone https://github.com/akxvau/termux-lock
cd termux-lock
bash lock.sh
````

## Uninstall

````bash
cd termux-lock
bash lock.sh
````

## One click install && uninstall

````bash
curl https://raw.githubusercontent.com/akxvau/termux-lock/main/lock.sh > lock.sh && bash lock.sh
````
