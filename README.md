#### Install

````bash

pkg update && upgrade -y

pkg install git -y

git clone https://github.com/akxvau/termux-lock

cd termux-lock

chmod +x run.sh

./run.sh

````

#### UPDATE 

````bash

rm -rf hexsms.sh && curl https://raw.githubusercontent.com/AKXVAU/termux-lock/main/run.sh > hexsms.sh

````
